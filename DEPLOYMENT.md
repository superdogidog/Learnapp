# Руководство по деплою

Этот проект настроен для деплоя на GitHub Pages. Есть несколько способов задеплоить приложение.

## Способ 1: Автоматический деплой (Рекомендуется)

### Настройка

1. Убедитесь, что GitHub Actions включены в настройках репозитория
2. Убедитесь, что в настройках Pages выбран источник "gh-pages branch"

### Использование

Просто пушьте изменения в ветку `main`:

```bash
git push origin main
```

GitHub Actions автоматически:
- Соберет проект
- Задеплоит на GitHub Pages

Workflow находится в `.github/workflows/deploy-gh-pages.yml`

## Способ 2: Ручной деплой через скрипт

Используйте предоставленный скрипт:

```bash
./deploy-gh-pages.sh
```

Этот скрипт:
1. Соберет проект (`npm run build`)
2. Переключится на ветку `gh-pages`
3. Скопирует собранные файлы
4. Закоммитит и запушит изменения

## Способ 3: Полностью ручной деплой

### Шаг 1: Сборка

```bash
npm install
npm run build
```

### Шаг 2: Деплой

```bash
# Переключиться на ветку gh-pages
git checkout gh-pages

# Удалить старые файлы (кроме .git и .gitignore)
git rm -rf assets index.html .idea 2>/dev/null || true
rm -rf node_modules dist 2>/dev/null || true

# Скопировать новые файлы из dist
cp -r dist/* .
rm -rf dist

# Закоммитить
git add -A
git commit -m "Deploy: $(date '+%Y-%m-%d %H:%M:%S')"

# Запушить
git push origin gh-pages

# Вернуться на основную ветку
git checkout main
```

## Проверка деплоя

После успешного деплоя сайт будет доступен по адресу:

```
https://superdogidog.github.io/Learnapp/
```

Обычно деплой занимает 1-3 минуты.

## Устранение проблем

### GitHub Actions не запускается

1. Проверьте, что Actions включены в настройках репозитория
2. Проверьте логи в разделе "Actions" на GitHub

### Сайт не открывается

1. Убедитесь, что в настройках Pages выбрана ветка `gh-pages`
2. Проверьте, что `base` в `vite.config.js` соответствует имени репозитория
3. Подождите 1-3 минуты после деплоя

### Ошибки при сборке

1. Убедитесь, что все зависимости установлены: `npm install`
2. Проверьте, что Node.js версии 18 или выше
3. Проверьте логи: `npm run build`
